== Installation ==
To use Thank Money Post, you will need:

* 	an installed and configured copy of WordPress version 3.8.x
*	FTP or SFTP access to your web host

= New Installations =

1.	Download the Thank Money Post archive in zip or gzipped tar format and
	extract the files on your computer. 

2.	Create a new directory named `thank_money_post` in the `wp-content/plugins`
	directory of your WordPress installation. Use an FTP or SFTP client to
	upload the contents of your Thank Money Post archive to the new directory
	that you just created on your web host.

3.	Log in to the WordPress Dashboard and activate the Thank Money Post plugin.

5.	Configure the plugin and add it to the plugin settings page.

== Frequently Asked Questions ==
Q: How to change the settings plug? 
R: Install the plugin settings on the Settings page.
